Syllabus and Policies

Installation

This is an introductory course to C programming. We will start from the very basic programming concept. You will need to install required text editor and compiler in order to code.

For Mac Users

  • Download TextMate
  • Download XCode from App store then install command line tools to run gcc.

For Window Users

  • Download mingGW
  • Download NotePad++

Textbook

We will be using Learn C the Hard way to reference in class exercise.
You can check it out here. Learn C The Hard Way

Quizzes

There will take home and in-class quizzes. Since everyone is very busy with work and other school assignments, I will try to minimize the amount of take home quizzes and focus more on in class exercises.

What you need

Please always bring your laptop to class. We will do alot of hands on code. This will help you understand the concept.

Grade

We will have exam based on your understanding toward the end of course, which will be determine regardless your commitment.

Attendent

It is your own free-will to learn or not to. I will not take any attendent, but choose it right and make up your mind if you are up for the challenge.
